# Learning Goals

This project exists to turn accumulated maker knowledge into something visible, useful, and shareable.

## What this project should teach

- Hardware basics
- Breadboarding
- Button inputs and pullups
- LED circuits and resistors
- Piezo buzzers and tones
- Non-blocking timing with `millis()`
- Arduino coding
- Soldering practice
- Schematic design
- PCB layout
- Custom PCB outlines
- Debugging hardware and software
- Signal detection and filtering
- Content planning for YouTube/TikTok

## Video 1 skills

| Skill | What teaches it |
|---|---|
| Breadboarding | Arduino Uno circuit |
| Digital input | Morse key button |
| Internal pullups | `INPUT_PULLUP` buttons |
| LED basics | Signal LED with resistor |
| Sound output | Piezo buzzer |
| Timing | Dot/dash detection with `millis()` |
| Debugging | Serial Monitor output |
| Basic firmware | Morse lookup and triggers |

## Video 2 skills

| Skill | What teaches it |
|---|---|
| Schematic design | Translating breadboard to circuit |
| PCB layout | Routing and component placement |
| Custom board outlines | Telegraph/telegram board shape |
| Silkscreen design | Labels, easter eggs, URL hints |
| Soldering | Building the PCB |
| Manufacturing | Gerbers and PCBWay order |

## Video 3 skills

| Skill | What teaches it |
|---|---|
| Analog input | Microphone or light sensor |
| Signal processing | Detecting tone/no-tone |
| Filtering | Ignoring noise |
| Display output | OLED or scrolling screen |
| Calibration | Sensitivity adjustments |
| Advanced firmware | Real external Morse decoding |

## Tone of the content

The project should be honest about the learning process. Mistakes, rewires, backwards LEDs, noisy inputs, and awkward first attempts are all part of the story.
