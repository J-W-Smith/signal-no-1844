# Enclosure

This folder will hold 3D printing files and notes for the Signal No. 1844 physical enclosure/backer.

## Design direction

The PCB should remain visible. The printed part should support the board, not hide it.

Possible printed parts:

- Thin backer plate
- Desktop display stand
- Protective battery cover
- Clip-on handle or grip
- Telegraph-style desk base

## First enclosure goal

Make a simple backer that:

- Protects solder joints
- Makes the board comfortable to hold
- Leaves the front PCB visible
- Provides room for the buzzer
- Provides room for the battery holder if used
- Allows access to programming/power connector

## Future enclosure ideas

- Angled display stand for filming
- Snap-fit back shell
- Heat-set insert screw mounts
- Decorative telegraph desk base
- Storage slot for jumper wires or cheat sheet
