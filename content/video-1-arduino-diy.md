# Video 1 — Arduino DIY Build

## Working title

**I Built a Tiny Telegraph with Beginner Electronics Parts**

## Core promise

Use an Arduino Uno, breadboard, button, LED, and buzzer to make a working Morse trainer.

## Build scope

- Arduino Uno R3
- Breadboard
- Main key button
- Signal LED
- Piezo buzzer
- Serial Monitor decode output
- SOS Easter egg
- Optional mode/mute/play buttons if time allows

## Video structure

1. Show the idea: tiny telegraph/Morse trainer.
2. Show the parts.
3. Wire button.
4. Add LED.
5. Add buzzer.
6. Code dot/dash timing.
7. Decode letters.
8. Trigger SOS flashback.
9. Tease the PCBWay revised version.

## Best hook

> I built a tiny telegraph with an Arduino, and if you tap the right code, it flashes a secret message back.
