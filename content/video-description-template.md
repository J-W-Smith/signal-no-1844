## Long YouTube Description

I’m building **Signal No. 1844**, a pocket telegraph / Morse code trainer project from scratch.

The plan is to start with a true DIY Arduino bench build, then turn the messy prototype into a custom PCB, and eventually upgrade it into a smarter version that can listen to real Morse beeps or flashes and decode them.

In this video, I’m working on:

- Arduino Uno bench prototyping
- Button input for Morse code
- LED and buzzer feedback
- Dot/dash timing with code
- Secret Morse code triggers
- Learning hardware, soldering, coding, and circuit design one step at a time

The goal is not to pretend I’m an expert. The goal is to learn in public, make the mistakes, document the process, and hopefully build something cool along the way.
